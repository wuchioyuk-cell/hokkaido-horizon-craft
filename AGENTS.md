## Brand Positioning

J-ONE Horizon / Hokkaido Horizon is a local Hokkaido travel studio and travel concierge.

We are not a luxury travel company. We are not a discount travel agency. We are not a traditional travel agency selling generic packaged tours.

We help travelers plan Hokkaido with local context: where to stay, how to move, which routes make sense, when snow matters, how the seasons change the journey, and how to avoid rushing through places that deserve time.

Core message:

`Hokkaido, Don't Rush.`

Business focus:

- Accommodation
- Transportation
- Airport transfers
- Seasonal journeys
- Private routes
- Ski and snow experiences
- Local support before and during travel

Snow is a primary business pillar and must stay visible.

## Brand Personality

Every page should feel:

- Quiet
- Warm
- Human
- Editorial
- Japanese
- Authentic
- Premium but approachable
- Local to Hokkaido
- Practical without feeling ordinary

The brand should earn trust through restraint, clarity, useful detail, and real photography. It should not try to impress with fake luxury language or loud sales design.

## Navigation

Primary navigation should use this structure:

- Stay
- Journeys
- Experiences
- Snow
- About
- Contact

Do not hide Snow inside a minor category. It is a primary pillar.

## Editing Rules

- Work one section at a time.
- Only modify the section explicitly requested.
- Do not redesign unrelated sections.
- Do not refactor unless explicitly requested.
- Do not create generic AI-looking landing pages.
- Preserve existing user or prior-agent changes unless the user explicitly asks to change them.
- If you need to change shared styles, confirm the impact is limited and consistent with the design system.
- If a requested change would make the site feel like luxury travel, discount travel, SaaS, or a generic travel agency, stop and explain why.

## Never Use

Do not introduce:

- Gold luxury styling
- Beige-heavy palettes
- Heavy gradients
- Glowing effects
- Tech startup UI
- SaaS landing page patterns
- Travel agency banner style
- Fake luxury language
- Hard-sell booking copy
- Dense card-heavy sections
- Unnecessary decorative graphics

## After Each Task

Report:

- Which files were modified
- Whether React/CSS files were touched
- Whether unrelated sections were untouched
- Whether verification was run
