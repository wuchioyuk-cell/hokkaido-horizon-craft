# Hokkaido Horizon Design Bible

## Role Of This Document

This design bible is the core source of truth for the J-ONE Horizon / Hokkaido Horizon website.

Every future design, copy, layout, and implementation change must respect this document first. If a requested change conflicts with this design bible, stop and explain the conflict before editing.

## Project

This is the active website project for J-ONE Horizon / Hokkaido Horizon.

The website introduces a local Hokkaido travel studio and travel concierge. It should help visitors understand that Hokkaido travel works best when accommodation, transportation, routes, seasonal timing, and experiences are planned together.

## Core Message

`Hokkaido, Don't Rush.`

This is not just a headline. It is the brand philosophy.

It means:

- Do not treat Hokkaido as a checklist.
- Do not rush through too many places.
- Respect distance, weather, snow, daylight, and personal energy.
- Plan where guests stay and how they move as one connected journey.
- Leave room for season, local rhythm, and quiet moments.

The website should feel like it is asking visitors to slow down, choose better, and travel with more care.

## Brand Positioning

J-ONE Horizon / Hokkaido Horizon is:

- A local Hokkaido travel studio
- A travel concierge
- A practical planning partner
- A connector between stays, transport, routes, seasonal journeys, and snow experiences
- A calm local guide for travelers who want Hokkaido to feel easier, slower, and more personal

J-ONE Horizon / Hokkaido Horizon is not:

- A luxury travel company
- A discount travel agency
- A traditional package-tour agency
- A luxury hotel brand
- A generic OTA storefront
- A tech startup
- A SaaS product
- A generic AI-generated travel landing page

## What We Specialize In

The website should repeatedly make these strengths clear:

- Accommodation planning
- Transportation planning
- Airport transfers
- Private charter and route design
- Seasonal journeys
- Ski and snow experiences
- Local support before and during travel

The value is not any one service in isolation. The value is how these services work together.

## Brand Personality

The brand should feel:

- Quiet
- Warm
- Human
- Editorial
- Japanese
- Authentic
- Premium but approachable
- Local
- Patient
- Trustworthy

The brand should not feel flashy, loud, corporate, fake, overly polished, or aggressively commercial.

## Design Principles

Use these principles to make decisions:

- Less is more.
- Content before decoration.
- Photography before graphics.
- Calm before excitement.
- Trust before selling.
- Local specificity before generic claims.
- Practical clarity before poetic vagueness.
- Whitespace before density.

If an element does not improve clarity, trust, rhythm, or emotional tone, remove it or simplify it.

## Visual Language

The site should use:

- Large typography
- Minimal interface
- Photography-first composition
- Lots of whitespace
- Subtle animation
- Editorial layouts
- Restrained color
- Clear navigation
- Quiet rhythm between sections

The strongest visual asset should be real-feeling Hokkaido photography: landscapes, roads, snow, stays, seasonal details, and human travel moments.

## Copy Direction

Copy should be calm, grounded, and specific.

Prefer language like:

- Local Hokkaido travel studio
- Travel concierge
- Where you stay
- How you move
- How the seasons unfold
- Routes, stays, transfers, and experiences planned together
- Travel at your own pace
- Snow travel shaped around timing, access, and comfort

Avoid language like:

- Luxury escape
- Once-in-a-lifetime indulgence
- Exclusive luxury experience
- Unbeatable deal
- Cheapest package
- Book now before it is gone
- World-class premium paradise
- Generic "discover hidden gems" copy without local detail

## Visual Avoid List

These should never appear:

- Gold luxury style
- Heavy gradients
- SaaS landing page style
- Tech startup UI
- Travel agency banner style
- Fake luxury language
- Glowing effects
- Heavy shadows
- Dense card grids
- Beige-heavy visual system
- Generic abstract decorations
- Stock-looking people pretending to travel
- Overly promotional booking panels

## Snow Pillar

Snow is not a seasonal afterthought. It is a primary business pillar.

Snow can include:

- Ski experiences
- Snow travel
- Winter route planning
- Ski resort access
- Snow lessons or support
- Winter transport and luggage considerations
- Timing and weather-aware planning

Navigation, homepage structure, and service copy should keep Snow visible.

## Trust Strategy

The site should build trust through:

- Local presence
- Practical planning detail
- Clear service categories
- Realistic expectations
- Calm copy
- Strong photography
- Human tone
- Clear inquiry path

It should not rely on exaggerated luxury language, urgency, discounts, or fake exclusivity.

## Workflow Rules

Before modifying any code, always read:

- `AGENTS.md`
- `docs/design-bible.md`
- `docs/design-system.md`
- `docs/homepage-structure.md`

Work one section at a time. Do not redesign unrelated sections. Do not overwrite the brand direction to chase a generic website trend.
